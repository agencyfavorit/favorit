# 🎨 Исправление градиентов - детали

## ✅ Что было исправлено:

### **Проблема:**
- Градиенты были слишком слабые и почти невидимые
- CSS переменные для орбов работали неправильно
- Недостаточная opacity и контраст

### **Решение:**

#### 1. **Усиленные орбы (gradient-orb)**
```css
/* До (не видно): */
opacity: 0.4
--orb-color: rgba(99, 102, 241, 0.6)

/* После (отлично видно): */
opacity: 0.5-0.6
background: radial-gradient(circle, rgba(99, 102, 241, 0.8) 0%, rgba(99, 102, 241, 0.4) 30%, transparent 70%)
```

**Цвета орбов:**
- **Orb 1** (синий): `rgba(99, 102, 241, 0.8)` → `0.6 opacity`
- **Orb 2** (розовый): `rgba(236, 72, 153, 0.7)` → `0.5 opacity`
- **Orb 3** (фиолетовый): `rgba(139, 92, 246, 0.7)` → `0.55 opacity`

#### 2. **Улучшенный hero-bg**
```css
background: 
    radial-gradient(circle at 15% 25%, rgba(99, 102, 241, 0.25) 0%, transparent 40%),
    radial-gradient(circle at 85% 65%, rgba(236, 72, 153, 0.2) 0%, transparent 40%),
    radial-gradient(circle at 45% 80%, rgba(139, 92, 246, 0.15) 0%, transparent 35%);
```

**Изменения:**
- Увеличена opacity с 0.05-0.15 до 0.15-0.25
- Изменена форма с ellipse на circle для равномерности
- Оптимизированы позиции точек свечения

#### 3. **Анимированный фон body**
```css
body::before {
    animation: bgPulse 15s ease-in-out infinite;
}

@keyframes bgPulse {
    0%, 100% { opacity: 1; }
    50% { opacity: 0.7; }
}
```

**Эффект:** Мягкое "дыхание" фона

#### 4. **Усиленные волны**
```css
opacity: 0.12 (было 0.08)
height: 180px (было 150px)

background: linear-gradient(90deg, 
    transparent 0%, 
    rgba(99, 102, 241, 0.4) 20%,  /* было 0.2 */
    transparent 40%, 
    rgba(139, 92, 246, 0.4) 60%,
    transparent 80%, 
    rgba(236, 72, 153, 0.4) 100%);
```

#### 5. **Декоративные элементы секций**
```css
section::before {
    background: radial-gradient(circle, rgba(99, 102, 241, 0.03) 0%, transparent 70%);
}

/* Чередование цветов */
section:nth-child(even)::before {
    background: radial-gradient(circle, rgba(236, 72, 153, 0.03) 0%, transparent 70%);
}
```

#### 6. **Разделительные линии**
```css
.services::after,
.team::after {
    height: 1px;
    background: linear-gradient(90deg, transparent, rgba(99, 102, 241, 0.2), transparent);
}
```

## ⚡ Оптимизация

### Производительность:
- `will-change: transform, opacity` - GPU acceleration
- `filter: blur(100px)` вместо 120px - меньше нагрузка
- Убрана анимация opacity из float (теперь статично)
- Используются transform для анимации (hardware accelerated)

### Размер:
- CSS: 1418 строк (~28KB)
- Общий проект: 84KB
- После gzip: ~20KB
